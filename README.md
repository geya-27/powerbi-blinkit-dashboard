# Blinkit-PowerBi-Dashboard
# 📊 Blinkit Power BI Dashboard

This repository contains an end-to-end Power BI project.

---

## 🚀 Project Overview

The goal of this project is to analyze Blinkit sales data and build an interactive dashboard using Power BI.
It demonstrates the complete workflow from data cleaning to visualization.

---

## 🧠 Key Learnings

* Data cleaning using Power Query
* Data modeling and relationships
* DAX calculations and measures
* Creating interactive dashboards
* Using filters, slicers, and KPIs

---

## 📊 Dashboard Features

* Sales performance overview
* Category-wise analysis
* Outlet location insights
* KPI cards (Total Sales, Avg Sales, etc.)
* Interactive filtering and drill-down

---

## 📁 Project Structure

* `data/` → Raw dataset used in the project
* `dashboard/` → Power BI (.pbix) file
* `images/` → Dashboard screenshots
* `docs/` → Notes and explanations

---

## ⚙️ Tools Used

* Microsoft Power BI
* Excel / CSV Dataset

---

## 📌 How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Explore the dashboard and visuals
